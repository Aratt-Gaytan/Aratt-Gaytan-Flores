¿Qué es GitHub?

GitHub es una plataforma de desarrollo colaborativo que se utiliza principalmente para alojar proyectos de software y facilitar el uso de Git, un sistema de control de versiones distribuido. GitHub ofrece una interfaz web que permite a los desarrolladores gestionar su código, colaborar en proyectos, y realizar un seguimiento de los cambios de manera eficiente.

GitHub permite a los usuarios crear "repositorios" donde pueden almacenar el código fuente de sus proyectos y compartirlo con otros. Los desarrolladores pueden colaborar en tiempo real, proponer cambios mediante "pull requests", y revisar el historial completo de modificaciones. GitHub también es muy utilizado para el desarrollo de proyectos de código abierto, lo que permite que miles de desarrolladores contribuyan a un mismo proyecto desde diferentes partes del mundo.

Fundadores de GitHub

GitHub fue fundado en 2008 por:

  • Tom Preston-Werner: Ingeniero de software y emprendedor, conocido por su participación en varios proyectos de código abierto y su liderazgo inicial en GitHub.
  • Chris Wanstrath: Programador y cofundador, quien fue CEO de GitHub antes de la adquisición por Microsoft.
  • PJ Hyett: Desarrollador de software que también formó parte del equipo fundador.
  • Scott Chacon: Contribuyó como cofundador y es conocido por su trabajo como escritor y conferencista sobre Git y GitHub.

En 2018, GitHub fue adquirido por Microsoft, pero sigue siendo una plataforma clave para el desarrollo de software en todo el mundo, con millones de desarrolladores y proyectos alojados.
